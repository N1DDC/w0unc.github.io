Motorola MDC1200

Many of our club members have Motorola commercial radios that use MDC1200. MDC1200 is a low-data rate system that allows for User ID, Selective Calling, Paging and Radio Check. IDs are transmitted at the beginning and end of transmission. IDs are based on the DTMF lettering of the last 4 characters of a call sign to create a semi-unique value. The club fleet ID is 862.

Kenwood FleetSync

Kenwood FleetSync is a message system that allows for indiviual transceiver identification and text messages. For the few members that run Kenwood equipment, our club uses FleetSync at 1200 baud. ID's are based on the same system as our MDC1200 ID's. The same 4 digit ID's are used in both systems.
